Relative Units

  • % : Relative to the parent elemet
  • em : Relative to the font-size to the parent element 2em meanns 2 times
  • rem : Font size to the root element
  • vw :Relative to 1% of the width of the viewport*
  • vw :Relative to 1% of the height of the viewport*
  • vmin : 1% viewport smaller dimension
  • vmin : 1% viewport larger dimension
